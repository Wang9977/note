

## nvm 

+ 作用 ： 管理多个node和npm版本
+ 安装： `npm install nvm -g` 
+ 使用
  + 查看 `nvm ls`
  + 切换版本 `nvm use 版本`
  + 添加 `nvm install 版本`
  + 删除 `nvm uninstall 版本`



## nrm

+ 作用： 管理多个npm源
+ 安装： `npm install nrm -g`
+ 使用
  + 查看 `nrm ls`
  + 切换版本 `nrm use 别名`
  + 添加源 `nrm add 别名 url`
  + 删除源 `nrm del 别名`
  + 测试所有源的响应时间 `nrm  test`
